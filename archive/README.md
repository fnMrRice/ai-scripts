# 脚本说明

- ## repack-archive.sh

### 脚本用途

该脚本用于批量将当前目录下的 `.gz` 压缩文件转换为 `.xz` 压缩文件。它会保持原始文件内容不变，仅改变压缩格式，通常 `.xz` 格式能提供更好的压缩比。

### 基本用法

#### 命令行示例

```bash
# 进入包含 .gz 文件的目录
cd /path/to/your/gz/files

# 运行脚本
./repack-archive.sh
```

#### 使脚本可执行

```bash
# 为脚本添加执行权限
chmod +x repack-archive.sh
```

### 注意事项

**依赖要求：**

- 系统需要安装 `gzip` 工具（用于解压 .gz 文件）
- 系统需要安装 `xz` 工具（用于创建 .xz 文件）
- 大多数 Linux 发行版默认包含这些工具

**权限要求：**

- 需要对目标目录有读写权限
- 脚本文件需要有执行权限（使用 `chmod +x` 设置）

**使用警告：**

- 脚本会在当前目录创建新的 `.xz` 文件，但不会自动删除原始 `.gz` 文件
- 请确保有足够的磁盘空间存储转换后的文件
- 建议在运行前备份重要数据
